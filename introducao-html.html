Conhecendo o HTML

Introdução

HTML é a linguagem responsável pela estrutura das páginas web. Ela foi criada por Tim Berners-Lee e é utilizada em praticamente todos os sites da internet.

Importância do HTML

O HTML é extremamente importante para o desenvolvimento web. Muitos desenvolvedores realmente começam seus estudos aprendendo HTML antes de CSS e JavaScript.

Estrutura de uma página

Uma página HTML normalmente possui elementos como html, head e body.

Elemento Body

O elemento body representa o conteúdo visível da página.

Elemento strong

A tag strong possui valor semântico e indica importância no conteúdo.

Aqui vai ficar a parte de ênfase do texto

Exemplo:

Anthony não é pra subir as escadas (sem o strong)

Anthony não é pra subir as escadas (com o strong). Então, eu coloquei o strong na palavra que eu quero dar mais ênfase, que nessa frase seria a palabra não

Elemento b

A tag b serve apenas para aplicar negrito visual.

O texto fica em negrito

Exemplo:

Eu amo anime, mas o meu favorito é Black Clover

Elemento em

O elemento em serve para dar ênfase a palavras ou frases.

O Texto fica dessa forma

Exemplo:

Eu nunca vou desitir. Essa frase foi dita por Asta no anime Black Clover

Elemento i

O elemento i aplica apenas um efeito visual em itálico.

Exemplo:

Eu amo passar meu tempo assistindo anime
