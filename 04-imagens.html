Conhecendo o elemento de imagem

Sobre as imagens

Trabalhar com imagens no HTML envolve muito mais do que apenas colocar uma foto na página.

Existem várias formas, técnicas e recursos para exibir, organizar, otmizar e adaptar imagens para diferentes dispositivos

Formatos mais comuns de imagens

  • JPEG/JPG: Fotos e imagens compactadas
  • PNG: Suporta transparência
  • WEBpFormato moderno e leve (também suporta transparência)
  • Gif: Animações simples
  • Avif: Excelente qualidade com um tamanho pequeno (também suporta transparência)

Sobre o elemento img

A forma mais comum de inserir imagens no HTML é usando a tag img

Exemplo prático

Trabalhando com links externos

Linsks externos = Links ou URL de sites

SRC (source/fonte) - Atributo responsável pelo caminho da imagem, ou seja, define onde está a imagem

ALT é um texto alternativo caso a imagem não carregue, então temos que colocar uma descrição bem precisa para a imagem

Trabalhando com imagens locais

ALT (alternative/alternativo) é o texto alternativo queserá apresentado caso a imagem não carrege

O atributo alt é MUITO IMPORTANTE

  • Acessibilidade
  • leitores de tela
  • SEO
  • Fallback

Ajustando o tamanho das imagens

HEIGHT - O atributo height serve para ajustar o tamanho da altura da imagem

WIDTH - O atributo Width serve para ajustar o tamanho da largura da imagem

Trabalhando com imagens responsivas

O picture é um elemento do HTML usado para imagens responsivas e adaptativas.

Essa tag permite o seguinte:

  • Trocar a imagem conforme o tamanho da tela
  • Melhoras a performance
  • Carregar imagens diferentes para mobile, tablet e desktop

Srcset - serve pra oferecer multiplas versões de uma mídia e deixar o navegadorescolher a mais adequada conforme tamanho da tela, formato suportado, largura disponível, performace.
