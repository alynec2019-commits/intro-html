Entendendo as listas HTML

As listas do HTML, servem para organizar informações relacionadas de forma estruturada e semântica

As listas são compostas por elementos chamados de LI.

Os LI (List Itrm/ Item da lista). As listas não possuem Limite de itens

Ordered List - OL

O elemento OL serve para criarmos uma lista ordenada, ou seja, com ordem, importancia

Normalmente aparece numerada

Listas de compras

  1. Banana
  2. Maçã
  3. Laranja
  4. Uva

Urnordered List

Serve para criar listas sem ordem específica. Normalmente aparece com bolinhas (bullets)

Exemplo rápido

  • Trabalhar
  • Visitar minha Tia
  • Vou no parque
