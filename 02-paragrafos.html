Elementos para citações

Elemento cite

Eu vou me tornar o Rei Mago

Esse elemento serve para indicar a fonte de uma citação ou o título de uma obra (Black Clover)

Exemplo:

Como disse Albert Einstein, A imaginação é mais importanteque o conhecimento.

Elemento q

Esse elemento q (citar) serve para marcar citações CURTAS dentro de um texto

Elemento Blockquote

O elemento blockquote (bloco de citação longa) serve para indicar uma citação em bloco, ou seja, um trecho de texto que foi retirado de outra fonte

Exemplo:
Nem todos os que vagueiam estão perdidos. O senhor dos Anéis
