Conhecendo o elemento de navegação

Definição de link

Um link é um elemento que permite navegar de uma página, documento ou recurso para outro.

No HTML, a gente usa os links para:

  • Abrir um site externo
  • Levar o usuário para outra página do site
  • Iniciar dowload de arquivos
  • Abrir aplicativos como e-mail e telefone

Conhecendo o elemento a

O elemento usado para criar links é a tag a (anchor)

href (HyperText Reference) - Referência de hipertexto

targuet - Define onde o conteúdo do link será aberto quando o usuário clicar nele

  • self: Valor padrão do navegador. Indica que o site será aberto na mesma aba
  • _blank: Abre uma nova aba ou janela dependendo do navegador
  • _parent: Usamos quando a página está dentro de um Frame/ifreme , ou seja, o link abre o frame no pai
  • _top: abre o link ocupando toda a janela do navegador, saindo de todos os frames.
